# Kartik Joshi — Finance & AI Portfolio

Professional portfolio website for Kartik Joshi, Financial Analyst & AI-Enabled Finance Professional.

## Live Site
🔗 [kartikjoshi-portfolio.vercel.app](https://kartikjoshi-portfolio.vercel.app)

## Features
- Professional dark finance/trading terminal aesthetic
- Live ticker with skills/expertise
- Full work history timeline
- 5 AI & Finance apps showcase with descriptions
- Education, Certifications, Contact sections
- Fully responsive (mobile-friendly)
- No frameworks — pure HTML/CSS/JS, zero dependencies

## Tech Stack
- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Playfair Display, DM Sans, DM Mono)
- Deployed on Vercel

## Deployment
This site is deployed automatically via Vercel on every push to `main`.

## Apps Featured
1. **FinPulse** — AI-powered FP&A forecasting platform
2. **Portfolio Rebalancer** — Risk-based allocation simulator
3. **AML Transaction Monitor** — Suspicious activity screening tool
4. **FP&A Budget Simulator** — Budget planning & variance analysis
5. **Credit Risk Analyzer** — Borrower credit evaluation tool
